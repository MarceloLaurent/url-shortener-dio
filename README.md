# url-shortener-dio

Projeto criado junto da DIO para construção de um Encurtador de URL.

## Rodando o projeto

- `npm install`
- `npm run build`
- `npm run dev`

Banco de dados utilizado:

- MongoDB

Tipo de conexão com o banco:

- Connect your application to your cluster using MongoDB's native drivers
